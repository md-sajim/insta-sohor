Javascript কি ভাবে কাজ করে ?

Javascript কি ভাবে কাজ করে তা বুঝার আগে বুঝতে হবে ওয়েব ব্রাউজার এবং DOM সম্পর্কে। ওয়েব ব্রাউজার একটি ওয়েব পেজ লোড করে,HTML Parser করে এর বিসয়বস্তু DOM(ডকুমেন্ট অবজেক্ট মডেল) নামে পরিচিত তৈরি করে।একই সাথে DOM Javascript কোডে ওয়েব পেজে একটি লাইভ ভিউ উপস্থাপন করে। ওয়েব পেজ তৈরির জন্য HTML ও CSS কে DOM দ্বারা একত্রিত করে,তার পর Javascript File এবং Inlinecode লোডে করে কিন্তু code টি Run করে না,HTML ও CSS এর জন্য অপেহ্মা করে।HTML ও CSS লোড হয়ে গেলে প্রয়োজন অনুযায় Javascript কার্যকর হয়। এর ফলে DOM Javascript code দ্বারা আপডেট হয় এবং ব্রাউজার দ্বারা রেন্ডার করা হয়। Javascript কাজ করার জন্য আদেশ টা গুরুপ্তপূর্ন। এবং Javascript উপর থেকে নিচর দিকে পোগরাম রিড করে।

Local Storage and Session Storage মধ্যে পার্থক্য ?

Local Storage and Session Storage মধ্যে প্রধান প্রধান পার্থক্য গুলোঃ
Local Storage

  • LocalStorage এর মধ্য়ে Data কে Permanent Store করা যায়
  • ব্রাউজার Close করলেও Data Delete হয় না
  • LocalStorage 10MB Memory নিয়ে কাজ করে

Session Storage

  • SessionStorage এর মাধ্য়মে Data কে Temperarily Store করা যায়
  • ব্রাউজার Close করলে Data Delete হয়ে যায়
  • SessionStorage 5MB Memory নিয়ে কাজ করে

Javascript কি ভাবে কাজ করে?

Javascript কি ভাবে কাজ করে তা বুঝার আগে বুঝতে হবে ওয়েব ব্রাউজার এবং DOM সম্পর্কে। ওয়েব ব্রাউজার একটি ওয়েব পেজ লোড করে,HTML Parser করে এর বিসয়বস্তু DOM(ডকুমেন্ট অবজেক্ট মডেল) নামে পরিচিত তৈরি করে।একই সাথে DOM Javascript কোডে ওয়েব পেজে একটি লাইভ ভিউ উপস্থাপন করে। ওয়েব পেজ তৈরির জন্য HTML ও CSS কে DOM দ্বারা একত্রিত করে,তার পর Javascript File এবং Inlinecode লোডে করে কিন্তু code টি Run করে না,HTML ও CSS এর জন্য অপেহ্মা করে।HTML ও CSS লোড হয়ে গেলে প্রয়োজন অনুযায় Javascript কার্যকর হয়। এর ফলে DOM Javascript code দ্বারা আপডেট হয় এবং ব্রাউজার দ্বারা রেন্ডার করা হয়। Javascript কাজ করার জন্য আদেশ টা গুরুপ্তপূর্ন।
